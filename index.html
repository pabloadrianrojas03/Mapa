Cómputo en la Nube y Servicios

Por: [Rojas Arriaga Pablo Adrian]

El cómputo en la nube ha revolucionado la forma en que las empresas y los individuos acceden y utilizan los recursos tecnológicos. Esta tecnología permite el acceso bajo demanda a recursos compartidos de computación a través de Internet, ofreciendo flexibilidad, escalabilidad y reducción de costos.

Tipos de Servicios en la Nube

IaaS (Infraestructura como Servicio)

Proporciona recursos de infraestructura fundamentales como servidores virtuales, almacenamiento y redes. Ejemplos:

  • Amazon Web Services (AWS) EC2
  • Microsoft Azure Virtual Machines
  • Google Compute Engine
PaaS (Plataforma como Servicio)

Ofrece plataformas que permiten a los desarrolladores crear aplicaciones sin preocuparse por la infraestructura subyacente. Ejemplos:

  • Google App Engine
  • Microsoft Azure App Services
  • Heroku
SaaS (Software como Servicio)

Proporciona aplicaciones completas listas para usar a través de Internet. Ejemplos:

  • Google Workspace
  • Microsoft 365
  • Salesforce

Patrones de Diseño en la Nube

Circuit Breaker

Patrón que detecta fallas y evita que se sature el sistema con solicitudes que probablemente fallarán. Actúa como un fusible eléctrico, "abriendo el circuito" cuando se detectan demasiados fallos.

Microservicios

Arquitectura que estructura una aplicación como una colección de servicios débilmente acoplados, altamente mantenibles y probables, organizados alrededor de capacidades empresariales.

Event Sourcing

Patrón que almacena todos los cambios en el estado de la aplicación como una secuencia de eventos, permitiendo reconstruir estados pasados y facilitando la auditoría.

CQRS

Patrón que separa las operaciones de lectura (queries) y escritura (commands) en modelos diferentes, optimizando el rendimiento, la escalabilidad y la seguridad.

Estándares en Servicios de Nube

Principales Estándares y Protocolos
Estándar Descripción Importancia
REST Arquitectura para sistemas distribuidos basada en HTTP Interoperabilidad entre servicios
OAuth 2.0 Estándar de autorización para APIs Seguridad en acceso a recursos
OpenAPI Especificación para documentar APIs RESTful Documentación consistente
ISO/IEC 27017 Estándar de seguridad para servicios en la nube Protección de datos
CDMI Interfaz estándar para almacenamiento en la nube Portabilidad de datos

Consideraciones sobre Estándares

La adopción de estándares en los servicios de nube es crucial para garantizar la interoperabilidad, seguridad y portabilidad de las aplicaciones. Organizaciones como ISO, NIST y la Cloud Security Alliance trabajan continuamente en el desarrollo y actualización de estos estándares para abordar los desafíos emergentes en el cómputo en la nube.
